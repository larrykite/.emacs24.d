+ My emacs 24 configuration. In the spirit of emacs-reboot:
http://avdi.org/devblog/2011/08/08/emacs-reboot-1-beginnings/.

Emacs-reboot provided the framework for creating a configuration from scratch
which I proceeded to make messy. Reorganization is definitely needed.

Features: python development with pymacs, pyflakes, ropemacs, etc., and other 
nice features.

Requires a file, pycheckers, somewhere in your system path, with these lines:

<<< Replace <shebang> with "#!/bin/bash" >>>

<shebang>/bin/bash

pyflakes "$1" 

pep8 --ignore=E221,E701,E202 --repeat "$1"

true
